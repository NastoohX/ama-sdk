Configuring the Package Feed¶

Distribution based package feeds allow for convenient and robust methods to update and upgrade relevant packages of the AMD AMA Video SDK. In order to be able to install the AMD AMA Video SDK packages from the package feed, point your package management client to the remote AMD AMA Video SDK package repository according to the instructions described below.

Ubuntu¶

Run the following commands on the system, where you wish to access the artifactory:

  1. Refer to lounge instructions, setting up the repositories - under Linux VM - for completing package feed setup.

  2. curl -u <lounge_email address> https://packages.xilinx.com/token

    Carefully, follow the instruction under APT section. Your output should look like the following:

    ...
################# APT ##################
# Create a new file under /etc/apt/auth.conf.d/, e.g. /etc/apt/auth.conf.d/xilinx.conf (make sure it is readonly by root), with the following content
#
machine packages.xilinx.com login xxxx@xxxx.com password xxxxxxxxxxxxxxxxxxxxxxx
#
# Create a new file under /etc/apt/sources.list.d/, e.g. /etc/apt/sources.list.d/xilinx.list with the following content.
# Make sure to change <CODE_NAME> with the OS code name, e.g. focal, jammy and etc.
#
deb [trusted=yes] https://packages.xilinx.com/artifactory/debian-packages/ <CODENAME> main
#

    , where CODENAME is the output of lsb_release -sc

Note

  1. Tokens obtaind via the above curl command are only valid for 48 hours.

  2. The 2nd step above, requires focal as the name of the distro.
